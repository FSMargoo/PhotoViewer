## 软件介绍
PhotoViewer 是一个简单的图片查看软件，使用 VuiLib 进行界面构造，总体只有百来行代码，非常适合大家学习~
## 操作指南
在软件中，可以通过左右方向键进行图片切换，并且可以用鼠标滚轮缩放图片。
## 支持的操作系统
Windows7 及以上
## 源码编译环境
* 编译目标平台：x64 x32
* 操作系统： Windows Vista 及以上
* API：EasyX2021730 & D2D9 及以上
